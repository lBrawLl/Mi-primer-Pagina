What is HTML?

HTML stands for Hyper Text Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of the elements that describe different types of vontent: pararapsd, link, headings, images, viedo, etc. web browsers uderstand HTML and render HTML code as websites.
